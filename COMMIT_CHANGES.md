# Изменения для коммита

## Исправления в команде UpdateCitiesExternalId

1. **Фильтрация только городов** - добавлена проверка `whereNotNull('region_id')` чтобы обрабатывать только города, а не регионы
2. **Улучшена логика извлечения ObjectId** - добавлена проверка длины и формата ObjectId, улучшено логирование
3. **Добавлена документация по отладке** - файл CITIES_EXTERNAL_ID_DEBUG.md

Для применения изменений на сервере нужно выполнить:
```bash
git pull
```

Затем повторить выполнение команды:
```bash
php artisan cities:update-external-id
```
